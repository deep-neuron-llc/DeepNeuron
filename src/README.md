## ⚙️ Project Setup

### 1. Clone the Starter

```bash
git clone https://github.com/deep-neuron-llc/deep_neuron_website.git
cd deep_neuron_website
```

### 2. Install Dependencies

With **npm**:

```bash
npm install
```

Or with **pnpm**:

```bash
pnpm install
```


## 🚀 Run the App

```bash
pnpm next dev
```
